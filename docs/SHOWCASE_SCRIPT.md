# Showcase Script

## 1. Opening narrative
- This is a **public demo** focused on transparency, humanitarian coordination and Pi ecosystem experimentation.
- The stack combines **Next.js**, **FastAPI**, **Pi RPC**, **Soroban-first architecture**, and a detector pipeline.

## 2. Suggested page order
1. `/`
   - explain the mission and product framing
2. `/launch`
   - explain readiness and current stage
3. `/dashboard`
   - show operational overview, map, hotspots and observability
4. `/hotspots`
   - show active regions
5. `/analytics`
   - show ranking, priority and executive comparisons
6. `/methodology`
   - explain scoring, evidence and current limits
7. `/proofs`
   - show registration trail and deployment source
8. `/donations`
   - show aggregate impact and funding narrative
9. `/transparency`
   - close with data sources, deployment and public status

## 3. Honest public language
Use terms like:
- public demo
- testnet
- evolving platform
- transparency-first prototype
- Soroban-aligned architecture

Avoid claiming:
- audited production readiness
- real humanitarian disbursement at scale
- official Pi Network endorsement unless formally granted

## 4. Technical closing
- API has health, public status, proofs, deployment and data-source endpoints.
- Dashboard is prepared for Vercel.
- API is prepared for Render/Railway.
- CI validates typecheck and production build.
